#  腾讯 C++ 代码规范

### 背景

经沟通，腾讯 C++ 代码规范将以业界普遍使用的Google代码规范为基础，经专家逐条评审并根据腾讯自身业务特性进行优化，最终评审结果将作为腾讯C++代码规范。
